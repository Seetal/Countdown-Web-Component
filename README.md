# Countdown Web Component

A simple web component that counts down a specified number of seconds.

## Usage

Link to the countdown-timer.js file.

Add the following html:

```html
<countdown-timer
  class="timer"
  id="timer-3"
  time="20"
  auto-start="true">
    <p class="time-remaining" data-timer role="timer"></p>
</countdown-timer>
```
